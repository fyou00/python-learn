# Integer

## Apa itu Integer?

Integer adalah bilangan bulat tanpa angka desimal.

Contoh:

```python
age = 20
temperature = -5
```

### Operasi

```python
10 + 5
10 - 5
10 * 5
10 / 5
10 // 3
10 % 3
10 ** 2
```

---

# String

## Apa itu String?

String adalah kumpulan karakter yang diapit tanda petik.

```python
name = "Fathur"
city = 'Lhokseumawe'
```

### Operasi yang dipelajari

- Concatenation
- Indexing
- Slicing
- String Methods
- f-string

Contoh:

```python
name = "Fathur"

print(name.upper())
print(name.lower())
print(name.replace("F", "M"))
```

# Penting

- Python bersifat dynamically typed.
- Gunakan `type()` untuk melihat tipe data.
- Gunakan `isinstance()` untuk mengecek tipe data.

```python
print(type(10))
print(type("Hello"))
print(type(True))
```
